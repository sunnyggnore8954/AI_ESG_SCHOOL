# 🌲 지역·연도별 산림 면적 데이터를 분석하여 미래 산림 면적 변화를 예측하고, 산림 보호와 관리에 활용하는 AI ESG 프로젝트

<br> <br>

## 📌 프로젝트 한눈에 보기 :
AI ESG School에서 산림파괴로 발생하는 환경 문제를 주제로 진행한 프로젝트입니다. 지역별 과거 산림 면적 데이터를 분석하여 미래 산림 면적 변화를 예측하고, 감소가 예상되는 지역의 산림 관리 필요성을 확인할 수 있는 인공지능 모델을 개발했습니다.

<br>

## 🚨 문제 정의 :
- 개발과 불법 벌채 등으로 산림 면적이 지속해서 감소하면서 탄소 배출량 증가, 산불, 산사태, 생물 다양성 감소와 같은 환경 문제가 발생하고 있습니다. 산림은 탄소를 흡수하고 생태계를 유지하는 중요한 역할을 하므로, 산림 감소가 계속되면 기후와 인간의 생활에도 직접적인 영향을 줄 수 있습니다.

- 산림 면적 감소는 장기간에 걸쳐 진행되기 때문에 현재 상태만 확인하는 것으로는 대응하기 어렵다는 한계가 있습니다. 이에 과거 산림 데이터를 바탕으로 미래의 변화를 예측하고, 산림 보호가 필요한 지역을 사전에 파악하는 방법을 구상했습니다.

<br>

## 💡 해결 아이디어 :
산림통계시스템에서 수집한 지역별·연도별 산림 면적 데이터를 활용하여, 미래 산림 면적을 예측하도록 설계했습니다.

ex) 지역별 산림 면적 데이터 수집 → 데이터 전처리
선형 회귀 모델 학습 → 미래 연도별 산림 면적 예측
산림 면적 감소 확인 → 식목 및 산림 관리 필요성 제시

사용자가 예측하고 싶은 지역을 입력하면 해당 지역의 과거 산림 면적 데이터를 모델이 학습하고, 미래 연도별 예상 산림 면적을 출력하도록 구성했습니다. 산림 감소가 예상되는 경우에는 해당 지역의 산림 관리와 식목 활동이 필요하다는 경고를 제공하도록 구현했습니다. <br>

<br>

## 🎯 프로젝트 목표 :
이 프로젝트는 인공지능을 활용해 산림 면적의 변화를 사전에 파악하고, 산림파괴로 발생할 수 있는 환경 문제를 예방하는 것을 목표로 했습니다.
(과거 산림 면적 데이터 분석 -> 미래 산림 면적 변화 예측 -> 산림 감소 가능 지역 파악 -> 식목 및 산림 보호 계획 수립 -> 지속 가능한 산림 생태계 유지) <br>

<br>

## ⚙️ 개발 과정:
1. 산림파괴 문제 조사
- 국내외 산림 면적 감소 현황과 산림파괴로 발생하는 탄소 배출, 산불, 산사태, 생물 다양성 감소 문제를 조사했습니다. 이를 통해 미래 산림 면적을 예측하고 관리에 활용하는 방향을 기획했습니다.


2. 산림 데이터 구성
- 산림통계시스템에서 지역별·연도별 산림 면적 데이터를 수집하고, 측정 연도·지역명·산림 면적으로 구성된 CSV 파일로 정리했습니다.
모델 학습 및 예측
- 연도와 산림 면적의 관계를 학습할 수 있도록 선형 회귀 모델을 구현했습니다. 학습용 데이터와 테스트용 데이터를 분리한 뒤, 사용자가 입력한 지역의 미래 산림 면적을 연도별로 예측하도록 구성했습니다.


3. 결과 출력 및 시각화
- 예측 기간의 연도별 산림 면적과 최고·최저·평균 면적을 출력하고, 실제 데이터와 예측 결과를 그래프로 시각화했습니다. 산림 면적 감소가 예상되는 경우에는 경고 메시지가 출력되는지도 확인했습니다. <br>

<br>

## 🛠 사용 기술 :
- Python - 전체 프로그램 개발 및 데이터 처리
- Pandas - CSV 데이터 불러오기 및 전처리
- NumPy - 연도와 산림 면적 데이터 배열 처리
- Scikit-learn - 선형 회귀 모델 학습 및 미래 면적 예측
- Matplotlib - 실제 산림 데이터와 예측 결과 시각화
- CSV - 지역별·연도별 산림 면적 데이터 저장 <br>

## 🌱 결과 및 배운 점 :
- 사용자가 지역을 입력하면 해당 지역의 미래 산림 면적을 연도별로 예측하는 프로그램을 구현했습니다. 이 과정에서 데이터를 모델에 적용하기 전에 연도와 지역별 데이터 형식을 정확하게 정리하는 전처리 과정이 중요하다는 점을 확인했습니다.

- 인공지능의 예측 결과를 단순히 출력하는 데서 끝내지 않고, 산림 감소 지역의 식목과 관리 계획에 활용할 수 있는 ESG 해결 방안으로 연결했습니다. 이를 통해 환경 문제를 해결하기 위해서는 기술 개발뿐만 아니라 예측 결과의 활용 방법까지 함께 설계해야 한다는 점을 배웠습니다.
